# Sbox-DDT-LAT

## Run

* cd /src/me/merteroglu
* Javac *.java
* Java me.merteroglu.Main

## Usage

* Enter the bit size (4 bits 16 digits)
* Enter the S-Box values. (Copy from simple.txt and paste all to console)
* Show the tables
